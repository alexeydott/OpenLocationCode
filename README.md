# OpenLocationCode
Delphi port for [Google](https://maps.google.com/pluscodes/) [Open Location Code](https://en.wikipedia.org/wiki/Open_Location_Code/).
The implementation is based on code from google [open-location-code](https://github.com/google/open-location-code/tree/main/c/) repository.
For ease of use a geo-calculator and methods for calculating neighboring cells have also been added.
